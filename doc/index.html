MakeGrid

The MakeGrid application makes az/alt grids to use with TUI for collecting pointing data. Download a MacOS application, or run from source in tcc/utils/makeGrid/runMakeGrid.py.

Instructions

  • Set the desired number of points and altitude range. You can usually leave the other settings alone.
  • Press Generate and wait while the grid is generated
  • Examine the grid to see if it is OK. If it has two points that are very near each other, then it is not OK. Press Generate to try again.
  • Once you like the grid, press Save to save it. TUI's Telescope:Pointing Data script looks for grid files in TUIAdditions/Grids, and you do not need to restart TUI to see new grid files (the script looks for grid files every time you click on the Grid menu).
  • You can also load existing grid files using Load, in case you want to examine the graph.
